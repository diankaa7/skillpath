# Настройка AI функционала для SkillPath

## Что добавлено:

✅ **Улучшенные рекомендации профессий** - AI анализирует ответы и дает более точные рекомендации
✅ **Персонализированные советы по навыкам** - индивидуальные рекомендации по развитию слабых сторон
✅ **Чат-бот консультант** - интерактивный помощник для вопросов о карьере
✅ **Анализ текстовых ответов** - глубокий анализ открытых вопросов для лучшего понимания личности

## Как это работает:

### 1. Без API ключа (текущий режим)
Система использует умную локальную логику, которая работает без подключения к внешним сервисам. Это позволяет:
- Получать базовые рекомендации профессий
- Видеть анализ навыков
- Использовать чат-бот с локальной логикой

### 2. С API ключом (для улучшенных результатов)

#### Вариант A: OpenAI API (рекомендуется)

1. Получите API ключ на https://platform.openai.com/api-keys
2. Добавьте в `ai-service.js`:

```javascript
// В конце файла ai-service.js или в script.js при загрузке страницы
aiService.setAPIKey('ваш-api-ключ-здесь');
```

#### Вариант B: Использование через Backend (безопаснее)

Создайте простой backend сервер (Node.js/Python), который будет:
- Хранить API ключи безопасно
- Обрабатывать запросы к AI
- Возвращать результаты на frontend

Пример структуры:
```
backend/
  server.js (или server.py)
  .env (с API ключами)
```

Frontend будет отправлять запросы на ваш сервер вместо прямого обращения к API.

## Использование:

### Для пользователей:
1. Пройдите тест (10 вопросов + 3 текстовых)
2. Получите AI-рекомендации профессии
3. Увидите анализ ваших ответов
4. Получите персональные советы по развитию навыков
5. Используйте чат-бот для дополнительных вопросов

### Для разработчиков:

#### Добавление нового AI провайдера:

В `ai-service.js` можно добавить поддержку других провайдеров:

```javascript
// Пример для Anthropic Claude
async callAnthropic(messages) {
  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': this.apiKey,
      'anthropic-version': '2023-06-01'
    },
    body: JSON.stringify({
      model: 'claude-3-sonnet-20240229',
      max_tokens: 1024,
      messages: messages
    })
  });
  // ...
}
```

#### Настройка модели:

```javascript
// В ai-service.js измените:
this.model = 'gpt-4'; // для лучших результатов (дороже)
// или
this.model = 'gpt-3.5-turbo'; // для экономии (текущая)
```

## Безопасность:

⚠️ **ВАЖНО**: Никогда не храните API ключи в клиентском коде!

Для production используйте:
1. Backend сервер для обработки AI запросов
2. Переменные окружения для хранения ключей
3. Rate limiting для защиты от злоупотреблений
4. Валидацию и санитизацию пользовательского ввода

## Стоимость:

- **OpenAI GPT-3.5-turbo**: ~$0.001-0.002 за запрос
- **OpenAI GPT-4**: ~$0.01-0.03 за запрос
- **Локальная логика**: Бесплатно (ограниченные возможности)

## Troubleshooting:

### AI не отвечает:
- Проверьте подключение к интернету
- Убедитесь, что API ключ правильный (если используется)
- Проверьте консоль браузера на ошибки
- Система автоматически переключится на локальную логику

### Медленные ответы:
- Используйте GPT-3.5-turbo вместо GPT-4
- Добавьте кэширование для частых запросов
- Оптимизируйте промпты (короче = быстрее)
